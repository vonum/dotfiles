# Dotfiles

Simple dotfiles containing:
- aliases
- zshrc
- vimrc
- tmux.conf
- prompt
- gitconfig

### Installation

```
git clone https://github.com/vonum/dotfiles ~/dotfiles

cd ~/dotfiles

./install
```
